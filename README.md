# HistoryJammiesITCS4230
Hop Through History: A Microgame Palooza
Created by Caeley Henderson, Bee Acree, Haley Genadio, and Alissa Davis

Hop Through History is a collection of minigames that progressively harder as the player explores more and more of these historical eras! Use one of three escape portals when failing a minigame. Keep winning minigames to increase your score and even earn a score multiplier!
Each minigame has its own control scheme, listed below.
Dino Disaster: Left and Right arrow keys OR A and D
Painting Patience: Left and Right arrow keys OR A and D
Various Vases: Mouse/Left Click
Cruising Cannonballs: W,A,S,D OR arrow keys to aim and SPACE to shoot

Cheat Codes:
Alt + D Starts Dino Disaster
Alt + C Starts Cruising Cannonballs
Alt + P Starts Painting Patience
Alt + V Starts Various Vases
Alt + R Restarts Game
Alt + H Goes to highscore/win screen
Alt + L Goes to lose screen
Alt + S Adds 100 points
Alt + M Adds 1 to multiplier
Alt + W Adds 1 to winStreak
Alt + K Adds 1 to portals/lives
